# Blinkit Data Dashboard

This repository contains a Power BI dashboard based on Blinkit data.

## 📁 Files Included

- `blinkit.pbix` – Power BI dashboard file
- `blinkit_Dashboard.pdf` – Exported view of the dashboard
- `BlinkIt Grocery Data.xlsx` – Source data file

## 🛠 Tools Used

- Microsoft Power BI
- Microsoft Excel

## 🚀 Usage

Open the `.pbix` file using Power BI Desktop to view or modify the dashboard.
